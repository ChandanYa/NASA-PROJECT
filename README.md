# NASA React App

This is a React app that fetches and displays data from NASA's APIs, including the Astronomy Picture of the Day (APOD) and Mars Rover photos.

## Features
- Display the Astronomy Picture of the Day.
- Launch a rocket with sound effects and animations.
- View Mars Rover photos.
